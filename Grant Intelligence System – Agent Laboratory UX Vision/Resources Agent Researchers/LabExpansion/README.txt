Welcome to basics:

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The base size of this tileset is 16px x 16 px.

“Land of pixels” is a work in progress asset project, so my main goal is to create a lot of tilesets, sprites, UI tiles that can be use as a robust and complete tileset. And so further developments will be made on nano banana pro to enhance the UI/UX.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
